## 2. 本ポリシーの適用範囲

本ポリシーは、当社が `griftai.org` で提供する Grift Beta に適用されます。  
当社のコーポレートサイトや他サービスに別途定めがある場合は、それぞれのポリシーが併せて適用されることがあります。

## 3. 取得する情報

当社は、本サービスの提供、改善、お問い合わせ対応のため、以下の情報を取得することがあります。

- お名前、メールアドレス、会社名、役職など、お問い合わせ時にユーザーが入力する情報
- お問い合わせ内容、相談内容、要件、案件情報
- ブラウザ種別、IP アドレス、端末情報、閲覧ページ、参照元、アクセス時刻などの利用情報
- Cookie 等の識別子
- GitHub 連携時に取得される repository 情報、organization 情報、メタデータ、開発活動に関する情報
- 本サービス上で入力、生成、保存された案件情報、見積情報、提案情報、設定情報

## 4. 利用目的

当社は、取得した情報を以下の目的で利用します。

- 本サービスの提供、運営、保守
- ユーザーからのお問い合わせ、資料請求、デモ依頼への対応
- 見積支援、提案支援、GitHub 実績分析、ベータ提供機能の実行
- 本サービスの改善、新機能検討、品質向上
- 利用状況の分析、障害調査、不正利用防止、セキュリティ対応
- 当社サービスに関する案内、連絡、重要なお知らせの送信
- 法令または契約上必要な対応

## 5. Cookie 等の利用

当社は、本サービスの利便性向上、アクセス解析、性能改善のために Cookie 等を利用することがあります。  
ユーザーはブラウザ設定により Cookie を無効化できますが、その場合、一部機能が利用できなくなる可能性があります。

## 6. アクセス解析

当社は、本サービスの利用状況を把握し改善するため、アクセス解析ツールを利用することがあります。  
解析ツールにより収集される情報には、IP アドレス、閲覧ページ、利用環境、遷移元などが含まれる場合があります。

## 7. GitHub 連携情報の取扱い

ユーザーが GitHub 連携を利用する場合、当社は本サービスの機能提供に必要な範囲で repository 情報や関連メタデータを取得・処理することがあります。  
当社は、連携情報を営業資料化、分析、可視化、提案支援のために利用することがあります。

当社は、連携情報のうち不要な生データを漫然と保存しないよう努め、必要な範囲でのみ取り扱います。

## 8. 案件情報、提案情報、生成物の取扱い

本サービスでは、案件概要、要件、見積、提案、判断メモなどの業務情報が入力される場合があります。  
当社は、これらを本サービス提供、改善、障害調査、サポート対応のために利用します。

ベータ期間中は、機能改善のためにログや生成結果を確認する場合があります。  
ただし、機密性の高い情報については、ユーザー自身の判断で入力範囲を調整してください。

## 9. テナント分離と匿名化方針

当社は、設計上、テナントごとの情報分離を重視しています。  
また、cross-tenant benchmark 等の構想に関しては、匿名化、集計、opt-in を前提に扱います。

raw text や直接識別子を含む情報を、無条件に横断利用する方針ではありません。

## 10. 学習利用に関する考え方

当社は、ユーザー情報や案件情報を無断で学習用途に利用する前提ではありません。  
分析、匿名化 benchmark、学習利用などが関係する場合は、別途の同意条件または設定を前提とする方針で運用します。

## 11. 第三者提供

当社は、次の場合を除き、ユーザーの個人情報を第三者に提供しません。

- ユーザー本人の同意がある場合
- 法令に基づく場合
- 人の生命、身体または財産の保護のために必要で、本人同意を得ることが困難な場合
- 業務委託先に、必要な範囲で取扱いを委託する場合

## 12. 委託

当社は、本サービス運営に必要な範囲で、クラウド基盤、アクセス解析、メール送信、決済、開発支援などを第三者へ委託することがあります。  
この場合、当社は委託先を適切に選定し、必要な監督を行います。

## 13. 安全管理措置

当社は、取得した情報について、漏えい、滅失、毀損、不正アクセス等を防止するため、合理的な安全管理措置を講じるよう努めます。  
ただし、インターネット通信およびクラウドサービスに絶対的な安全性は存在しないことをご理解ください。

## 14. 保有個人情報の開示等

ユーザーは、法令の定めに基づき、自己の個人情報について開示、訂正、削除、利用停止等を求めることができます。  
希望する場合は、当社問い合わせ窓口までご連絡ください。

## 15. 未成年者の利用

未成年者が本サービスを利用する場合は、必要に応じて法定代理人の同意を得たうえで利用してください。

## 16. ポリシーの変更

当社は、必要に応じて本ポリシーを変更することがあります。  
重要な変更を行う場合は、本サービス上または当社が適切と判断する方法で周知します。
